# Job Market Analytics Dashboard 2023

## Overview
An end-to-end data analytics project analyzing 32,000+ job postings 
to help job seekers make data-driven career decisions. Built using 
Excel Power Query for data cleaning and Power BI for interactive 
visualization.

## Tools & Technologies
- Excel Power Query — data cleaning & transformation
- Power Pivot — data modeling & relationships
- DAX — calculated measures & KPIs
- Power BI Desktop — interactive dashboard

## Data Pipeline
Raw CSV → Power Query Cleaning → Star Schema Model → DAX Measures → Power BI Dashboard

## Dashboard Features
- Interactive slicers (Role, Country, Date, Job Type)
- Cross-filtering across all visuals
- Dynamic chart highlighting
- Quarterly job posting trend
- Top 10 skills analysis
