# 📊 Job Market Analytics Dashboard 2023

> An end-to-end data analytics project analyzing 32,000+ job postings to help job seekers make data-driven career decisions. Built using Excel Power Query for data cleaning and Power BI for interactive visualization.

---

## 📌 Dataset Source

Dataset sourced from ** https://github.com/lukebarousse/Excel_Data_Analytics_Course/tree/main **

---

## 🎯 Business Questions Answered

- Which job roles offer the highest average salaries?
- Do remote jobs pay more than onsite roles?
- Which countries have the highest job demand?
- Which skills are most in demand across all roles?
- Does degree requirement affect salary levels?
- How did job postings trend across 2023?

---

## 🗺️ Data Pipeline

```
Raw CSV → Power Query Cleaning → Star Schema Model → DAX Measures → Power BI Dashboard
```


---

## 📊 Dashboard Features

- ✅ Interactive slicers — Role, Country, Date Range, Job Type
- ✅ Cross-filtering across all visuals
- ✅ Dynamic chart highlighting — top value auto-highlighted
- ✅ Dynamic dashboard title — updates with slicer selection
- ✅ Quarterly job posting trend line chart
- ✅ Top 10 skills analysis with relationship across tables
- ✅ Remote vs Onsite salary & distribution comparison
- ✅ Role demand vs salary combo chart
